### 문제<br/>
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.
<br/><br/>

### 입력<br/>
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.
<br/><br/>

### 출력<br/>
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.
<br/><br/>

### 예제 입력 1<br/>
```
2143
```
### 예제 출력 1<br/>
```
4321
```
<br/>

### 예제 입력 2<br/>
```
999998999
```
### 예제 출력 2<br/>
```
999999998
```
<br/>

### 예제 입력 3<br/>
```
61423
```
### 예제 출력 3<br/>
```
64321
```
<br/>

### 예제 입력 4<br/>
```
500613009
```
### 예제 출력 4<br/>
``` 
965310000
```
