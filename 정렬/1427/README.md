## ![1427번 소트인사이드](https://www.acmicpc.net/problem/1427) <br>


### 문제 <br>
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

### 입력 <br>
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

### 출력 <br>
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.
<br>

### 예제 입력 1
```
2143
```
<br>

### 예제 출력 1
``` 
4321
```
<br>

### 예제 입력 2
```
999998999
```
<br>

### 예제 출력 2
```
999999998
```
<br>

### 예제 입력 3
```
61423
```
<br>

### 예제 출력 3
```
64321
```
